# Aqara_David - Aqara Bridge for Home Assistant

基于Aqara开放平台，通过云端API进行设备控制以及订阅。

## 特点

- 🚀 在原版基础上增强，支持更多设备类型
- 🔒 已清理所有敏感信息，可安全分享
- 📚 完整的中文文档和使用指南
- ✨ 新增Button、Number、Select设备支持
- 🛠️ 扩展的设备映射

## 安装要求

- Home Assistant 2024.1.0 或更高版本
- Python 3.11 或更高版本
- Aqara开发者账号（需申请appId、appKey、keyId）

## 配置步骤

1. 在Aqara开放平台申请开发者账号
2. 获取appId、appKey、keyId
3. 在Home Assistant中添加集成
4. 输入Aqara账号和开发者凭证
5. 完成配置后，设备会自动导入

详细说明请查看[完整文档](https://github.com/lpf479/Aqara_David)
